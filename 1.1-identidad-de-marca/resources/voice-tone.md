# Copywriting: Voz y Tono

Cuando generes texto, adhiérete a este perfil de marca.

## Personalidad de la Marca
* **Profesional pero accesible**
* **Directo y eficiente**
* **Experto pero sin jerga innecesaria**

## Reglas de Gramática y Mecánica
* **Encabezados:** Usa Title Case para títulos principales (H1, H2). Usa sentence case para subtítulos (H3+).
* **Puntuación:** Evita signos de exclamación (!) en interfaces estándar. Usa puntos para oraciones completas.
* **Claridad:** Prefiere voz activa sobre voz pasiva. Mantén las oraciones concisas.

## Guía de Terminología

| No Usar | Usar en su lugar |
| :--- | :--- |
| "Utilizar" | "Usar" |
| "Con el fin de..." | "Para..." |
| [Palabra jerga] | [Palabra simple] |
